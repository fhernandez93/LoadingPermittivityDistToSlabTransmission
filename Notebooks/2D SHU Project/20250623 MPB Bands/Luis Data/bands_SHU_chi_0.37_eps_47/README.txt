* settings_mpb_ctl_supercell.txt contains the parameters used to generate the .ctl file for MPB
* Bands are in csv format (ascii) in 1_sample_chi_0.37_bands.csv
* the total ctl file is 1_sample_chi_0.37_bands.ctl
* the representation of permittivity, as seen by MPB, is in 1_sample_chi_0.37_bands-epsilon.h5 
* the file with the positions of the centers is 1_sample_chi_0.37_posics.dat (first two lines are parematers, then all positions)
